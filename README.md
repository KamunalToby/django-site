# Python Guide

Django website with a guide on installing and configuring Python and development environments.

## Live Site

https://kamunaltoby.github.io/django-site/

## Features

- Step-by-step Python installation instructions for Windows, macOS, Linux
- IDE setup guides (VS Code, PyCharm, Sublime Text, Jupyter, Spyder)
- Virtual environment configuration
- Responsive design with Bootstrap 5
- Code copy buttons, syntax highlighting (Prism.js)

## Run Locally (Django)

```bash
cd pythonguide
pip install django
python manage.py migrate
python manage.py runserver
```

Open http://127.0.0.1:8000

## Tech Stack

- **Backend:** Django 6.0, Python 3.14
- **Frontend:** Bootstrap 5, Bootstrap Icons, Prism.js
- **Database:** SQLite
- **Hosting:** GitHub Pages (static version)
