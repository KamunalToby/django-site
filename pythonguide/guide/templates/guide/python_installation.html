Установка Python

Важно: Рекомендуется устанавливать Python версии 3.8 или выше для лучшей совместимости с современными библиотеками.

Установка на Windows

1Скачайте установщик

Перейдите на официальный сайт python.org и скачайте последнюю версию Python для Windows.

Прямая ссылка: python.org/downloads/
2Запустите установщик

Запустите скачанный .exe файл. В появившемся окне обязательно отметьте:

  • Add Python to PATH - это позволит запускать Python из командной строки
  • Install launcher for all users (опционально)
Внимание: Обязательно отметьте "Add Python to PATH"!
3Выберите компоненты

На экране "Optional Features" рекомендуется установить:

  • Documentation
  • pip (менеджер пакетов)
  • tcl/tk and IDLE (встроенная IDE)
  • Python test suite
  • py launcher
4Выберите путь установки

Оставьте путь по умолчанию или выберите свой. Рекомендуется путь без пробелов и кириллицы.

5Завершение установки

Дождитесь завершения установки. Нажмите "Close" когда процесс завершится.

Установка на macOS

1Проверьте установленную версию

macOS обычно поставляется с предустановленным Python 2.7. Проверьте версию в терминале:

python --version
python3 --version
2Установите Homebrew (рекомендуется)

Homebrew - лучший способ управления Python на macOS:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
3Установите Python через Homebrew
brew install python

Это установит последнюю версию Python 3 и pip.

4Альтернативный способ (официальный установщик)

Скачайте .pkg файл с python.org и установите как обычное приложение macOS.

Установка на Linux

1Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip python3-venv
2CentOS/RHEL/Fedora
# Для Fedora
sudo dnf install python3 python3-pip

# Для CentOS/RHEL
sudo yum install python3 python3-pip
3Arch Linux
sudo pacman -S python python-pip

Проверка установки

После установки проверьте, что всё работает правильно:

# Проверка версии Python
python --version
python3 --version

# Проверка pip
pip --version
pip3 --version

# Запуск интерактивной оболочки
python
Успешная установка: Если команды выше работают без ошибок, Python установлен правильно!
Команды для проверки
Проверка версии: 
python --version
Запуск Python: 
python
Проверка pip: 
pip --version
Частые проблемы
Python не найден:

Убедитесь что "Add Python to PATH" был отмечен при установке.

Проблемы с правами:

Запустите командную строку от имени администратора.

Старая версия:

Удалите старые версии Python перед установкой новой.
