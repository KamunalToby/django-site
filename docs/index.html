Установка Python

Важно: Рекомендуется устанавливать Python версии 3.8 или выше для лучшей совместимости с современными библиотеками.

Установка на Windows

1Скачайте установщик

Перейдите на официальный сайт python.org и скачайте последнюю версию Python для Windows.

Прямая ссылка: python.org/downloads/
2Запустите установщик

Запустите скачанный .exe файл. В появившемся окне обязательно отметьте:

  • Add Python to PATH - это позволит запускать Python из командной строки
  • Install launcher for all users (опционально)
Внимание: Обязательно отметьте "Add Python to PATH"!
3Выберите компоненты

На экране "Optional Features" рекомендуется установить:

  • Documentation
  • pip (менеджер пакетов)
  • tcl/tk and IDLE (встроенная IDE)
  • Python test suite
  • py launcher
4Выберите путь установки

Оставьте путь по умолчанию или выберите свой. Рекомендуется путь без пробелов и кириллицы.

5Завершение установки

Дождитесь завершения установки. Нажмите "Close" когда процесс завершится.

Установка на macOS

1Проверьте установленную версию

macOS обычно поставляется с предустановленным Python 2.7. Проверьте версию в терминале:

python --version
python3 --version
2Установите Homebrew (рекомендуется)

Homebrew - лучший способ управления Python на macOS:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
3Установите Python через Homebrew
brew install python

Это установит последнюю версию Python 3 и pip.

4Альтернативный способ (официальный установщик)

Скачайте .pkg файл с python.org и установите как обычное приложение macOS.

Установка на Linux

1Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip python3-venv
2CentOS/RHEL/Fedora
# Для Fedora
sudo dnf install python3 python3-pip

# Для CentOS/RHEL
sudo yum install python3 python3-pip
3Arch Linux
sudo pacman -S python python-pip

Проверка установки

После установки проверьте, что всё работает правильно:

# Проверка версии Python
python --version
python3 --version

# Проверка pip
pip --version
pip3 --version

# Запуск интерактивной оболочки
python
Успешная установка: Если команды выше работают без ошибок, Python установлен правильно!
Команды для проверки
Проверка версии:
python --version
Запуск Python:
python
Проверка pip:
pip --version
Частые проблемы
Python не найден:

Убедитесь что "Add Python to PATH" был отмечен при установке.

Проблемы с правами:

Запустите командную строку от имени администратора.

Старая версия:

Удалите старые версии Python перед установкой новой.

Настройка среды разработки

Совет: Выбор IDE зависит от ваших предпочтений. Начните с VS Code - он бесплатный и мощный.

Visual Studio Code (Рекомендуется)

1Установка VS Code

Скачайте VS Code с официального сайта:

Скачать VS Code
2Установка расширений

Откройте VS Code и установите следующие расширения через вкладку Extensions (Ctrl+Shift+X):

  • Python от Microsoft - основное расширение для Python
  • Pylance - интеллектуальная поддержка Python
  • Python Docstring Generator - генерация документации
  • GitLens - улучшенная работа с Git
  • Bracket Pair Colorizer - подсветка скобок
3Настройка Python в VS Code
  1. Откройте папку с проектом в VS Code
  2. Нажмите Ctrl+Shift+P и выберите "Python: Select Interpreter"
  3. Выберите установленную версию Python
  4. VS Code автоматически предложит создать виртуальное окружение
4Настройка форматирования

Установите и настройте форматирование кода:

# Установка форматеров
pip install black autopep8 flake8
Настройки VS Code (Ctrl+,):
  • "python.formatting.provider": "black"
  • "editor.formatOnSave": true
  • "python.linting.enabled": true

PyCharm

1Выбор версии

PyCharm доступен в двух версиях:

  • Community Edition - бесплатная, для обычной разработки
  • Professional Edition - платная, с дополнительными инструментами

Для начинающих рекомендуется Community Edition.

2Установка

Скачайте с jetbrains.com/pycharm и установите как обычное приложение.

3Настройка проекта
  1. Запустите PyCharm и создайте новый проект
  2. Выберите интерпретатор Python
  3. PyCharm предложит создать виртуальное окружение
  4. Настройте параметры проекта в Settings
4Полезные плагины
  • .ignore - поддержка .gitignore файлов
  • Key Promoter X - изучение горячих клавиш
  • GitToolBox - улучшенная работа с Git

Другие популярные IDE

Sublime Text

Легкий и быстрый редактор с расширением Package Control

  • Установите Package Control
  • Добавьте пакеты: Anaconda, SublimeLinter
Jupyter Notebook

Идеально для научных вычислений и анализа данных

  • Установка: pip install jupyter
  • Запуск: jupyter notebook
Spyder

Научная IDE, похожая на MATLAB

  • Входит в состав Anaconda
  • Отличная для анализа данных
IDLE

Встроенная IDE Python, поставляется вместе с Python

  • Не требует установки
  • Подходит для первых шагов

Настройка виртуального окружения

Виртуальное окружение - обязательная практика для Python разработки:

Создание виртуального окружения:
# Создание окружения
python -m venv myenv

# Активация (Windows)
myenv\Scripts\activate

# Активация (macOS/Linux)
source myenv/bin/activate

# Деактивация
deactivate
Установка пакетов:
# Установка пакетов
pip install django requests numpy

# Сохранение зависимостей
pip freeze > requirements.txt

# Установка из файла
pip install -r requirements.txt
